# Development Guide

## Project Structure

```text
skillpack/
├── src/                 # TypeScript source for the CLI
│   ├── cli.ts
│   ├── commands/
│   └── core/
├── runtime/             # Runtime bundled into the zip output
│   ├── server/          # Express backend
│   ├── web/             # Web chat UI
│   ├── scripts/         # Startup scripts
│   └── README.md
├── app.json             # Current pack config, generated by create
└── skills/              # Installed skills
```

## Build the CLI

```bash
npm install
npm run build
```

## Test the CLI

After building from the project root, you can run:

```bash
# Test interactive creation and write output to a directory
npm run create -- output
```

## Debug the Runtime Bundle

The `runtime/` directory contains the code that is packaged into the zip and can be run independently for debugging:

```bash
cd runtime/server
npm install
# PACK_ROOT should point to a directory containing app.json and skills/
PACK_ROOT=/path/to/your/pack-dir node index.js
PACK_ROOT=../../output node index.js
```

Open [http://127.0.0.1:26313](http://127.0.0.1:26313) in your browser to debug the web UI.
